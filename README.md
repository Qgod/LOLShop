# Shop
## 客户端：
1、客户浏览所有鲜花产品；    
2、客户查看某一产品的详情；  
3、可以将某产品捡入购物车；  
4、从详情页返回到首页；  
5、单击购物按钮直接添加至购物车；  
6、删除购物项；  
7、清空购物车；  
8、提交订单；  
9、修改数量；  
10、继续选购；  
11、填写购买人信息；  
12、订单信息确认；  
13、提交订单（已登录用户）；  
14、订单查询（单前用户）；  
15、用户登录；  

##  服务端：
1、产品信息查询；  
2、新增产品信息；  
3、产品信息维护；  
4、产品下架；  
5、查询所有用户信息；  
6、订单查询；  
7、处理订单；
